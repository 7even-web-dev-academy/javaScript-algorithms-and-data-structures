# Declarar variables de cadena de caracteres (string)

Previamente usó el siguiente código para declarar una variable:

~~~
var miNombre;
~~~

Pero también puede declarar una variable de cadena o **string**, como esta:

~~~
var miNombre = "tu nombre";
~~~

**"tu nombre"** se denomina literal de cadena o **string** en ingles. Un **string**, o cadena, es una serie de cero o más caracteres entre comillas simples o dobles.
